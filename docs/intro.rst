Introduction
============

What is Ironman?
----------------

Ironman is a general purpose software toolbox to be run
on L1Calo hardware with embedded processors (SoCs).

Look how easy it is to use:::

    import ironman
    # Get your stuff done
    ironman.engage()

Features
--------

- Be awesome
- Make things faster

Installation
------------

Install ironman by running:

    pip install ironman
